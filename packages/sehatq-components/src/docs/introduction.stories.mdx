# Introduction

Sehatq web monorepo design using turborepo features. There ares some packages and apps:

- `sehatq`: SehatQ web application powered by [Next.js](https://nextjs.org)
- `tokoq`: TokoQ web application powered by [Next.js](https://nextjs.org)
- `@sehatq/components`: docs, user interfaces and domain-spesific features powered by [Storybook](https://storybook.js.org/) and [tsup](https://tsup.egoist.sh/)
- `@sehatq/constants`: shared Constants
- `@sehatq/types`: shared non generic type
- `@sehatq/utils`: shared React, generic type and function utilities
- `@sehatq/tsconfig`: shared `tsconfig.json`s used throughout the monorepo
- `eslint-preset-sehatq`: ESLint preset
